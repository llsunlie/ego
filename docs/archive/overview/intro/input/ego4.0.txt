# ego · 产品总结（v4）
你迷茫的那一刻，你会打开谁。

## 产品是什么
ego 是一个记得你的独白空间。

你写下此刻的想法，ego 从你过去说过的话里找到一句回应你。不是建议，不是安慰——是让你看见，原来你自己早就想过这件事。

不是日记，不是 AI 聊天，不是心理测试。是你和你自己，跨越时间的对话。

- 产品名：ego
- 一句话：写下此刻，遇见过去的自己。

## 产品三层结构
```
┌──────────────────────────────────────┐
│  此刻                                │
│  写下 → 回声 → 发现 → 继续想        │
│  一次体验的价值：这一刻被照见了      │
├──────────────────────────────────────┤
│  过往                                │
│  每一次说出口的，都留在这里          │
│  一周使用的价值：我的轨迹在这儿      │
├──────────────────────────────────────┤
│  星图                                │
│  散落的自己，慢慢连成星座            │
│  长期的价值：我的生命在形成某种形状  │
└──────────────────────────────────────┘
```

## 页面功能
### 页面一 · 此刻（主屏）
#### 画面
- 深色星空背景，随机分布的小星星在闪烁
- 中央一团柔软的、不规则形变的呼吸光团（内部有光核脉冲）
- 光团周围有三颗记忆光点（暖金 / 冷蓝 / 柔紫），在缓慢漂浮
- 周围还有寄存的小光点在轨道上转动
- 下方一行淡灰引导："有什么想说的吗"
- 按钮："写下此刻"

#### 记忆光点（盲盒）
- 每次打开主屏随机展示 3 颗，内容从历史话语中随机抽取
- 点击展开为一封精致的信封——里面是过去某天写过的一句话

#### 写字流程
1. 点"写下此刻" → 光缩小上移，输入区在原地滑出（不跳页）
2. placeholder："随便说点什么，这里听着……"
3. 点"先到这"手动提交

#### 回声
- 提交后，浮出一张卡：过去某天的一句相似的话
- 上方小标签："你之前也说过类似的"——帮用户理解这张卡是什么
- 下方："之前的你还说过 ›"——展开 2-3 条候选回声，用户可以切换

#### "✦ 我发现"
- 回声下方的观察卡，用第二人称对用户说
- 例："两次里，你用的词不一样——一次是'等'，一次是'别扭'。同一件事，好像有了新的名字。"

#### 三个出口
- "顺着再想想"（主按钮，金色）→ 输入区重新滑出，继续写。写完仍会触发新回声，可无限延续，用户主导深度
- "✦ 收进星图"（辅动作）→ 把这次完整的对话结构（Trace）保存为一颗星。触发仪式感动效：卡片发光 → 三圈金色涟漪 → 一颗光点沿贝塞尔曲线飞向星图 tab → 拖尾 → 星屑炸开 → tab 金色脉冲
- "嗯，先这样"（退出，极轻灰字）

"收进星图"收的是什么：一个完整的 Trace（一次痕迹），包含：用户这次的话 + 回声 + "我发现" + 如果用户"顺着再想想"继续写的所有延续。不是单条话，是一整次对话结构。

### 页面二 · 过往
页面说明："每一次说出口的，都留在这里"

#### 画面
- 按月分段的时间线
- 每一条 = 日期 + 内容摘要
- 已被联结进星座的条目右上角显示 ✦ 已联结
- 点击展开看完整内容

### 页面三 · 星图
页面说明："散落的自己，慢慢连成星座"

#### 画面
- 深空背景（像星云），柔和过渡
- 星星更大，每颗有不同的闪烁节奏，此起彼伏
- 星座之间的连线在做呼吸动画
- 标签有 backdrop-filter 和金色光晕脉冲

#### 三种星星状态
| 状态 | 视觉 | 含义 |
| ---- | ---- | ---- |
| 已成型星座 | 亮星 + 连线 + 标签名 | 你反复说过的主题，已被识别 |
| 正在成型 | 弱光脉冲 + "隐约有什么…" | 2颗相关的话还没凑够形成主题 |
| 孤星 | 独立弱光 | 一次性的话，在等同伴 |

点击星座 → 进入独立的主题详情页（push 进去，左上角返回箭头）：

#### 详情页三块内容
① ✦ 我发现
- 金色卡片，ego 对这个主题的观察
- 例："三次都是'他不说话，我也不说'。第一次你用的是'等'，第二次是'内耗'，第三次是'别扭'——同一件事，你叫它的名字在变。"
- 底部可折叠："主题里说过的话 ›" → 展开原话列表

② 和那时的自己说说话
- 紫色卡片列表，最多展示 3 条，超出折叠
- 点击进入对话模式：

**对话机制**
- AI 扮演"那段时间的你"，用第一人称回应
- 三层材料：原话是地基（思想灵魂永不伪造）、延伸是血肉（用那时的语气和想法灵动回应，可以反问）、标注是诚实（每句回复底部标"参考了 X月X日 前后"）
- AI 可以反问用户、可以自省（"我那时候还没想明白"），但不能捏造那时没想过的话题
- 如果用户问了那时候没想过的事，回答"这个我那时候没想过，要不要你说给我听？"
- 开场白示例："我那时候写过这样一句——'其实我没有真的生气，我只是想让他先开口。' 你想跟那时的我聊什么？"

③ ✦ 我想和你聊聊
- 金色卡片，3 个不同角度的话题引子
- 每个引子锚定一句原话 + 顺着它提一个具体问题
- 例："顺着 2月11日：你说'我只是想让他先开口'——那'先开口'对你到底意味着什么？"
- 点击 → 自动引用话题回到主屏输入区，用户接着写
