# About EGO — Design Spec

**Date:** 2026-06-09
**Status:** Design approved

## Overview

Add an "About EGO" section to the existing setting page, displaying the app version sourced from git tags.

## Scope

- Front-end only — no proto or backend changes
- New generated file: `client/lib/core/version.dart`
- Modified file: `client/lib/features/setting/setting_page.dart`
- New Makefile target: `make version`

## Design

### Page Layout (after change)

```
SettingPage
├── AppBar（「设置」）
├── 账号信息区
│   ├── 标签「账号信息」（灰色 13px）
│   ├── 手机号行：左侧「手机号」标签 + 右侧脱敏值
│   └── 注册时间行：左侧「注册时间」标签 + 右侧格式化日期
├── 关于区                          ← NEW
│   ├── 标签「关于」（灰色 13px）
│   └── 版本行：左侧「版本」标签 + 右侧 version 值
└── 退出登录按钮
```

### Spacing

- 账号信息区 → 关于标签：32px gap
- 关于标签 → 版本行：16px gap
- 版本行 → 退出登录：48px gap

### UI Spec

- "关于" section label: `AppColors.textHint` (`#5A5A7A`), 13px, same style as "账号信息" label
- Version row: `_infoRow('版本', appVersion)` — reuses existing `_infoRow` helper
  - Left label: `AppColors.textHint`, 15px
  - Right value: `AppColors.textPrimary`, 15px

### Version Source

```
git describe --tags  →  client/lib/core/version.dart
```

Generated file content:
```dart
// Generated by `make version`. Do not edit.
const String appVersion = 'v0.1.9';
```

- On a tagged commit: `v0.1.9`
- On a non-tagged commit: `v0.1.9-17-g611db8a`
- The file is committed to git — it reflects the state at build time
- `make version` regenerates it on demand

## Implementation

### Files

| File | Action | Description |
|------|--------|-------------|
| `client/lib/core/version.dart` | Create | Generated version constant |
| `client/lib/features/setting/setting_page.dart` | Modify | Import version, add "关于" section |
| `Makefile` | Modify | Add `version` target |

### Makefile

```makefile
.PHONY: version
version:
	@echo "// Generated by \`make version\`. Do not edit." > client/lib/core/version.dart
	@echo "const String appVersion = '$$(git describe --tags --always)';" >> client/lib/core/version.dart
```

### Code Change in setting_page.dart

After the account info rows (`_infoRow('注册时间', ...)`) and its `SizedBox(height: 48)`, insert:

```dart
const SizedBox(height: 32),
const Text(
  '关于',
  style: TextStyle(
    color: AppColors.textHint,
    fontSize: 13,
  ),
),
const SizedBox(height: 16),
_infoRow('版本', appVersion),
const SizedBox(height: 48),
```

Remove the existing `SizedBox(height: 48)` before the logout button.

Add import: `import '../../core/version.dart';`
