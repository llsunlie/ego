# starmap Architecture

Bounded context: Starmap.

Responsibilities:

- Stash Trace as Star.
- Cluster Stars into Constellations.
- Maintain Constellation status.
- Generate constellation-level Insight.
- Generate PastSelfCard and TopicPrompt assets.
- Serve starmap overview and constellation detail.

Primary use cases:

- StashTrace
- ClusterStar
- ListConstellations
- GetConstellation
- RebuildConstellationAssets
